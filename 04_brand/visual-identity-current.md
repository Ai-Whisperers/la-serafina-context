## Hipótesis de trabajo

AIREANA y La Serafina tienen **identidad visual básica pero funcional**, no altamente profesionalizada. El sitio web parece ser un WordPress o similar con un theme estándar. La foto principal de portada y los íconos son consistentes.

Lo que falta:

1. **Sistema de diseño** — guía de uso de colores, tipografía, logos
2. **Jerarquía clara entre sub-marcas** — ¿cuándo usar el logo de AIREANA vs el de La Serafina?
3. **Manual de marca** — para que aliados puedan producir material coherente
4. **Plantillas para redes sociales** — Instagram, Facebook, etc.
5. **Identidad del Festival de Cine** separada del resto

## Recomendación para Ai-Whisperers

Si se aprueba el proyecto de rediseño:

1. **Fase 0 — Descubrimiento visual (1 semana)**
   - Capturar todo el material visual existente
   - Identificar inconsistencias
   - Entrevistar a AIREANA sobre historia de la marca

2. **Fase 1 — Manual mínimo viable (2 semanas)**
   - Logo system (variantes claro/oscuro, AIREANA vs La Serafina)
   - Paleta de 5 colores máximo
   - 2 tipografías (headline + body)
   - Iconografía básica para servicios
   - Tono de voz (ver `tone-of-voice.md`)

3. **Fase 2 — Sistema de plantillas (3 semanas)**
   - Templates para Instagram stories + posts
   - Templates para Facebook
   - Templates para flyers (PDF descargable)
   - Email signature
